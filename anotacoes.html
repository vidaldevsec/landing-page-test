Versionamento Git e Git Ignore

O que é Git Ignore

  • Arquivo para configurar quais arquivos/pastas o Git deve ignorar ao commitar
  • Evita enviar arquivos sensíveis para o repositório remoto (ex: variáveis de ambiente)
  • Evita enviar pastas muito grandes desnecessariamente (ex: node_modules)

Como usar

  • Criar um arquivo .gitignore na raiz do projeto
  • Listar os arquivos/pastas a serem ignorados (um por linha)

Exemplos

node_modules/.envconfig.php
  • Ao commitar, os arquivos/pastas listados no .gitignore não serão enviados

Vantagens

  • Maior segurança (não expõe dados sensíveis)
  • Menos uso de espaço no repositório remoto
  • Commits mais limpos e com apenas arquivos relevantes
