Git log

Importância

  • Permite ver e pesquisar o histórico de ações realizadas no repositório
  • Mostra informações como:
    • Posição atual (branch/head)
    • Autor e data do commit
    • Mensagem do commit
    • Identificação (hash) do commit

Comandos

  • git log --oneline: resume a exibição do commit em uma linha
  • git log --stat: exibe estatísticas dos arquivos alterados e quantidade de linhas
  • git log -p: exibe alterações mais detalhadas dentro de cada commit
  • git log --author=: pesquisa commits de determinado autor
  • git log --grep=: pesquisa commits que contenham determinado texto
  • git log --graph: exibe commits em formato gráfico de linha do tempo

Observações

  • Comandos podem ser combinados (ex: git log --oneline --author=João)
  • Permite identificar diferença entre branch local e remote
  • Importante para navegar entre commits depois
